# 🌍 GeoSmart Nexus

GeoSmart Nexus is an intelligent geospatial platform that combines GIS, AI, and Cloud technologies to provide interactive mapping, spatial analysis, and data visualization capabilities.

## 🚀 Tech Stack

- **Frontend:** Angular, MapLibre GL JS
- **Backend:** Spring Boot
- **GIS:** PostGIS, GeoServer, Martin
- **Data:** OpenStreetMap
- **DevOps:** Docker, Kubernetes, GitHub Actions

## ✨ Features

- Interactive maps
- Points of Interest management
- Spatial search and filtering
- Geolocation and routing
- Dashboards and reporting
- Multilingual support
- Scalable cloud-native architecture
